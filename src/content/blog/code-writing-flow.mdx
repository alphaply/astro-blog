在线写作工具能解决一部分问题，但也会带来更多状态、权限和保存链路。

现在的方向更简单：

1. 内容放在仓库里。
2. 文章使用 MDX。
3. 复杂展示封装成组件。
4. GitHub 和 Vercel 负责版本记录与部署。

这样维护成本更低，也更适合长期迁移旧博客内容。
